Getting started with Maven

Add dinject and dinject-controller dependencies.

```xml io.dinject dinject 1.5 io.dinject dinject-controller 1.5 ```

Java APT

For Java we add the APT Annotation processors dinject-generator and javalin-generator. We typically use provided scope for these APT processors.

```xml io.dinject dinject-generator 1.5 provided io.dinject javalin-generator 1.5 provided ```

Kotlin KAPT

For Kotlin KAPT the easiest thing to do is to use a maven tile that registers the KAPT processors with the Kotlin compiler.
