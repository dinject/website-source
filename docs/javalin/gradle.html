Getting started with Gradle

See the example at: examples/javalin-gradle-java-basic/build.gradle

Add dinject and dinject-controller as compile dependencies.

```groovy dependencies { ... compile('io.dinject:dinject:1.9') compile('io.dinject:dinject-controller:1.6') ... } ```

APT - dinject-generator

Add both javalin-generator and dinject-generator as a annotation processors.

```groovy dependencies { ... annotationProcessor('io.dinject:dinject-generator:1.9') annotationProcessor('io.dinject:javalin-generator:1.6') ... } ```

Kotlin KAPT - dinject-generator

See example at: https://github.com/dinject/examples/blob/master/basic-di-kotlin-maven/pom.xml

For use with Kotlin we register dinject-generator as a KAPT processor to the Kotlin compiler.

```groovy dependencies { ... kapt('io.dinject:dinject-generator:1.9') kapt('io.dinject:javalin-generator:1.6') ... } ``` <@next "Controller" "controller"/>
